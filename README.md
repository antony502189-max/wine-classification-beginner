# Классификация сортов вина

Небольшой учебный ML-проект для первого знакомства с классификацией. Модель получает 13 числовых характеристик вина и определяет один из трёх классов.

## Что вы изучите

- разделение данных на обучающую и тестовую выборки;
- объединение масштабирования и модели в `Pipeline`;
- обучение логистической регрессии;
- оценку качества с помощью accuracy и confusion matrix;
- сохранение модели и получение новых предсказаний.

## Быстрый старт

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python train.py
```

После запуска в папке `artifacts` появятся:

- `wine_model.joblib` — обученная модель;
- `metrics.json` — метрики;
- `confusion_matrix.png` — матрица ошибок.

## Предсказание

Передайте 13 характеристик через пробел:

```bash
python predict.py --features 13.2 2.7 2.4 18.5 100 2.6 2.8 0.3 1.8 5.0 1.0 3.0 1000
```
